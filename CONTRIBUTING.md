# 参与贡献 | Contributing

欢迎对 **Moonbrake** 产生兴趣！我们非常感谢你花时间来帮助改进这个项目。为了确保协作的质量与高效，请在提交贡献前阅读以下指引。

*Welcome to **Moonbrake**! We truly appreciate your interest in improving this project. To ensure high-quality collaboration, please read the following guidelines before making any contributions.*

## 💡 如何贡献 | How to Contribute

### 1. 提交 Issue | Report Issues
如果你发现了 Bug 或有建议，请通过 GitHub Issue 反馈。
*If you find a bug or have a feature suggestion, please open a GitHub Issue.*

### 2. 提交 Pull Request | Pull Requests
如果你想直接修改代码：
1. Fork 本仓库。
2. 创建新分支 (`git checkout -b feature/name`)。
3. 提交更改并发送 Pull Request。
*If you want to contribute code:*
1. *Fork the repository.*
2. *Create a new feature branch.*
3. *Submit a Pull Request.*

## ⚖️ 协议与规范 | Guidelines & License

### 1. 版权署名 (Mandatory Attribution)
**必须标注原作者 (IIclementineII) 并提供指向本 [原始仓库](https://github.com/IIclementineII/moonbrake) 的链接。**
*Must include clear attribution to the original author (IIclementineII) and a link back to this [original repository](https://github.com/IIclementineII/moonbrake).*

### 2. 代码风格 (Code Style)
*   保持代码简洁，添加必要注释。
*   *Maintain clean code and include necessary comments.*
